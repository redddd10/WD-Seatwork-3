Headers and Footers

‎ ‎ ‎

This activity aims to test the learner's ability to code a website using CSS and HTML. In this website, header and footer designs (static and animated) can be viewed. These designs aim to check the students design and formatting skills. Lastly, traversal between pages is implemented in this website as a means to build multi-page websites.
